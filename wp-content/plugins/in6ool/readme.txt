管理画面ダッシュボードや投稿での各ウィジェット、アーカイブページの表示を制御します。

== Description ==

* **PHP 5.5 未満の場合は動作しません。**
* 管理画面ダッシュボードや投稿での各ウィジェット、アーカイブページの表示を制御します。
* 管理画面フッターのテキストをカスタマイズします。
* ヘッダーに挿入される割と不要なものを削除します。
* フィードでの著者名を非表示にします。

== Installation ==

1. *プラグイン* メニュー配下の *新規追加* をクリックしてください。
2. *in6ool* で検索してください。
3. *in6ool* をインストールしてください。
4. *in6ool* を有効化してください。
5. *設定* メニュー配下の *in6ool* より設定をご確認ください。
