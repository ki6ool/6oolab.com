== Description ==
プラグインを有効化した後、表示させたい箇所に
`<?php if (class_exists('ZundokoKiyoshi')) new ZundokoKiyoshi();?>`
と書きます。

* 元ネタ: https://twitter.com/kumiromilk/status/707437861881180160
* まとめ: http://qiita.com/shunsugai@github/items/971a15461de29563bf90

== Installation ==

1. *プラグイン* メニュー配下の *新規追加* をクリックしてください。
2. *ZundokoKiyoshi* で検索してください。
3. *ZundokoKiyoshi* をインストールしてください。
4. *ZundokoKiyoshi* を有効化してください。
